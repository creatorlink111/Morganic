Why teams choose Morganic

  • Symbolic, compact syntax designed for readable logic in fewer tokens.
  • Three runtime implementations (Python, Rust, Node.js) kept in feature sync.
  • Built-in support for typed lists, loops, reusable functions, enums, and terminal graphing.

Syntax at a glance

Variables + loops

[sum]=^0^:4(0,5000){[sum]=|`sum+1|}:1([sum])

Functions + calls

#echo'msg.£'#{1(&msg)}:
#echo £hello from function

Conditionals

[value]=^120^:
2([value]..^120^){1(£exact match)}

Lists + indexing

[nums]=l(i)<^1^,^2^,^3^>:
[nums]~[nums]@^1^:
1([nums])
