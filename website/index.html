A concise language for expressive scripting, rapid iteration, and high-performance runtime options.

Explore performance

Why teams choose Morganic

  • Symbolic, compact syntax designed for readable logic in fewer tokens.
  • Dual runtime implementations in Python and Rust for flexibility and speed.
  • Built-in support for lists, loops, typed values, reusable functions, and terminal graphing.

Syntax at a glance

Variables + loops

[sum]=^0^:4(0,5000){[sum]=|`sum+1|}:1([sum])

Functions

{addOne}(x){
  [result]=|`x+1|
}:1(`result)

Conditionals

?(`value>100){
  :1("high")
}*{
  :1("ok")
}

Graphing

[points]=^1,5,4,8,6^
:5(`points)
